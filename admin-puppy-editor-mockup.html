Technical Implementation Notes

Thumbnail Component

  • • Uses `image_url` field for the main thumbnail
  • • Photos stored in `photos` JSONB array
  • • Videos stored in `video_url` field
  • • Drag-to-reorder for gallery images

Preview Function

  • • Opens `puppy-detail.html?id={puppyId}`
  • • Can use iframe for inline preview
  • • Or window.open() for new tab
  • • Responsive preview modes (desktop/tablet/mobile)

AI Generation

  • • Uses Claude API to analyze photos
  • • Generates: description, personality, best suited for
  • • Can regenerate individual fields
  • • Falls back to templates if API unavailable

Database Fields

  • • `image_url` - Main thumbnail
  • • `photos` - Gallery array
  • • `video_url` - Featured video
  • • `description` - AI-generated text
  • • `personality` - Traits list
